== Description ==

Se state utilizzando wordpress su [altervista](http://it.altervista.org "altervista") grazie a questo plugin potrete attivare in modo semplice il circuito pubblicitario JSToolTip, che vi permettera' di aumentare gli altercents e gli euro guadagnati con il vostro sito.

Non dovrete fare altro che scaricare il plugin ed attivarlo, semplice no? :-)

Tramite la pagina di gestione del plugin sar&agrave; possibile definire alcune parole per le quali non saranno visualizzate pubblicit&agrave;

[Pagina di Supporto](http://gerasimone.altervista.org/2009/jstooltip-4-altervista-wordpress-plugin "Pagina di Supporto")

== Installation ==

- Scaricate il plugin
- Copiatelo nella cartella wp_content/themes 
- Attivatelo dall'area di gestione del vostro wordpress
- (Facoltativo) Andate nella pagina Impostazioni > JSTooltip e configurate il plugin
